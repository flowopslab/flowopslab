# n8n Automation Starter

This is a compact scope checklist for a $50 same-day n8n workflow sprint.

## Good Fit

- One webhook or form intake flow
- Lead routing into Google Sheets, Airtable, Telegram, email, or an API
- One cleanup or validation step
- One notification with the fields a human needs to act
- One tested trigger-to-output path

## First-Scope Questions

1. What starts the workflow?
2. What destination should receive the result?
3. What fields are required?
4. What should happen when a field is missing or invalid?
5. What does a successful test look like?
6. Who should receive notifications?

## Same-Day Deliverable

- One workflow design or n8n JSON where appropriate
- One example input and expected output
- Setup notes
- Handoff notes and failure cases
