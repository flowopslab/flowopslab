# Landing Page Teardown Template

FlowOps Lab uses this compact format for $25-$50 same-day landing page reviews.

## Inputs

- Page URL
- Target buyer
- Primary action
- Current traffic source if known
- Current conversion problem if known

## Review Sections

1. Offer clarity: can a stranger explain what is sold in 10 seconds?
2. Buyer fit: does the page name the right user and pain?
3. Proof: are there public examples, screenshots, testimonials, or numbers?
4. CTA: is the next action obvious, low-friction, and repeated at the right spots?
5. Objection handling: price, trust, delivery time, and risk reversal.
6. Mobile scan: does the first viewport carry the offer and CTA?

## Same-Day Deliverable

- Short diagnosis
- Revised hero copy
- CTA rewrite
- Section-by-section fixes
- Three priority changes to make first
