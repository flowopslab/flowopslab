# Data Cleanup Checklist

Use this before sending a cleanup request.

## Include

- File type: CSV, XLSX, Google Sheet export, or other
- Approximate row count
- Column names
- What is broken or messy
- What the final file should look like

## Do Not Include Initially

- Passwords
- Private keys
- Full customer records
- Government IDs
- Payment card data
- Medical or highly sensitive personal data

Send a redacted sample first if the data is sensitive.
